Anabel's idea is to create a web page with a title, a guide style and a button. The code begins with the declaration of the document type and language. Then, in the section <head>, metadata such as the character set and the initial scale are established. An inline style is also defined that sets the body font, the primary color, and the styles for the button.

In <body> it, the main title is used with the “color-primary” class to apply the blue color defined in the style. A button is also created with the “button” class that inherits the styles defined in the inline style and changes its background color when you hover the cursor over it.
